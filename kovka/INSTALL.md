# Kovka — Инструкция по запуску

## Быстрый старт

### 1. Установка темы

```
wp-content/
└── themes/
    └── kovka/   ← скопировать эту папку сюда
```

WP Admin → Внешний вид → Темы → Активировать **Kovka**

При активации тема автоматически создаст страницы:
`/vorota/` `/zabory/` `/lestnitsy/` `/mebel/` `/dekor/` `/art/`
`/calculator/` `/portfolio/` `/about/` `/contacts/` `/blog/`

---

WP Admin → Страницы → каждая страница → «Атрибуты страницы» → Шаблон:

---

### 3. Установить ACF Pro

- Купить лицензию: https://www.advancedcustomfields.com/pro/
- Установить плагин: WP Admin → Плагины → Добавить новый → загрузить ZIP
- Импортировать поля: ACF → Инструменты → Импорт → выбрать `inc/acf-import.json`

> Без ACF Pro тема работает, но поля (цена, гарантия, галерея и т.д.) не отображаются.

---

### 4. Импортировать демо-контент (опционально)

WP Admin → Инструменты → Импорт → WordPress → выбрать `inc/demo-content.xml`

- 2 портфолио-работы
- 3 отзыва (со звёздами)

## Наполнение контентом

### Добавить изделие в каталог

WP Admin → Изделия → Добавить новое:
- Заголовок: название изделия
- Текст: подробное описание
- Категория (справа): выбрать нужную
- Материал: выбрать или добавить
- Миниатюра: главное фото

В блоке **«Характеристики изделия»** (ACF):
- Цена от / до
- Металл, покрытие, срок, гарантия
- Галерея (до 20 фото)
- Хит продаж ✓

### Добавить работу в портфолио

WP Admin → Работы → Добавить новую:
- Заголовок: краткое название проекта
- Описание: история проекта
- Категория: тип изделия

В блоке **«Данные работы»**:
- Город, год, бюджет, срок (дней)
- Галерея готовой работы
- Показывать на главной ✓ (для топ-5 на главной)

### Добавить отзыв

WP Admin → Отзывы → Добавить новый:
- Заголовок: имя клиента
- Текст: текст отзыва

В блоке **«Данные отзыва»**:
- Рейтинг (1–5)
- Город, источник (Google/Яндекс/Avito)
- Фото клиента (аватар)
- Показывать на главной ✓

### Добавить сотрудника

WP Admin → Команда → Добавить нового:
- Заголовок: ФИО
- Должность, стаж, специализация
- Фото (квадратное, минимум 400×400 px)

### Добавить FAQ

WP Admin → FAQ → Добавить новый:
- Заголовок: вопрос
- Текст: ответ

## Структура файлов темы

```
kovka/
├── style.css                    # Декларация темы + весь CSS
├── functions.php                # CPT, ACF, Schema.org, AJAX
├── header.php                   # Шапка, навигация, бургер
├── footer.php                   # Подвал, модалка, WhatsApp
├── front-page.php               # Главная (18 блоков)
├── page.php                     # Стандартная страница
├── single.php                   # Статья блога
├── single-kv_product.php        # Карточка товара
├── archive-kv_product.php       # Каталог с фильтром
├── index.php                    # Fallback
├── 404.php                      # Страница ошибки
├── page-templates/
│   ├── category.php             # Страница категории (6 шт)
│   ├── calculator.php           # Калькулятор (5 панелей)
│   ├── portfolio.php            # Портфолио с фильтром
│   ├── about.php                # О нас
│   └── contacts.php             # Контакты + форма + карта
├── assets/
│   └── js/
│       ├── main.js              # Вся интерактивность
│       └── calculator.js        # Логика калькулятора
└── inc/
    ├── acf-import.json          # Импорт групп полей ACF Pro
    └── demo-content.xml         # Демо-контент (WXR)
```

## Плагины

**Обязательные:**
- ACF Pro (поля изделий, команды, отзывов)

**Рекомендуемые:**
- Yoast SEO или Rank Math — SEO-мета для страниц
- WP Rocket или LiteSpeed Cache — кэширование
- Smush или WebP Express — оптимизация изображений
- WPForms Lite — резервная форма обратной связи
- UpdraftPlus — резервные копии

**Не нужны** (всё встроено в тему):
- Contact Form 7 (есть своя AJAX-форма)
- Breadcrumb NavXT (хлебные крошки встроены)
- Schema Pro (Schema.org генерируется автоматически)

## Настройка цен в калькуляторе

Базовые цены находятся в двух местах:

**1. Быстрый калькулятор на главной** — `assets/js/main.js`, строка ~185:
```js
var basePrices = {
  gates:     18000,   // ₽ за 1 м²
  fence:     4500,    // ₽ за 1 пог.м (1 м высоты)
  stairs:    22000,   // ₽ за 1 ступень
  railing:   6500,    // ₽ за 1 пог.м
  furniture: 35000,   // ₽ за единицу
  decor:     12000,   // ₽ за единицу
};
```

**2. Серверный AJAX-расчёт** — `functions.php`, функция `kv_ajax_calc()`:
```php
$prices = [
  'gates'     => 18000,
  'fence'     => 4500,
  'stairs'    => 22000,
  'railing'   => 6500,
  'furniture' => 35000,
  'decor'     => 12000,
];
```

**3. Полный калькулятор** — `assets/js/calculator.js`: цены в `<option value="...">` каждой панели.

## Решение проблем

**Страницы возвращают 404:**
→ WP Admin → Настройки → Постоянные ссылки → Сохранить (пересоздаёт `.htaccess`)

**Поля ACF не отображаются:**
→ Убедитесь, что ACF Pro активирован и `inc/acf-import.json` импортирован

**Форма не отправляется:**
→ Проверьте, что `wp_nonce_field` выводится в форме (нужен PHP)
→ Проверьте консоль браузера на ошибки JS

**WhatsApp-кнопка не появляется:**
→ Заполните поле «WhatsApp номер» в Настройках темы (только цифры: 79001234567)

**Карта не отображается:**
→ В поле «Ссылка на карту» вставьте `src` из iframe-кода Яндекс/Google карт (только URL, без тегов)
